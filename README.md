# QA Demo - Gerenciamento de Imóveis

Aplicação web simples em TypeScript e React para estudo prático de QA.

## Rodar localmente

```bash
cd qa-demo
npm install
npm run dev
```

Acesse `http://localhost:4173`.

## O que contém

- lista de imóveis com dados fake
- filtro de busca por título, endereço, proprietário ou status
- painel de detalhes do imóvel selecionado
- UI simples para execução de testes de QA
